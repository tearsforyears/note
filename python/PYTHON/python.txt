专用下划线标识符

　　_xxx 不用'from module import *'导入

　　__xxx 类中私有变量名

特殊意义的标识符 
_foo表示不能直接访问类 要通过类提供的接口来访问
不能用 from xxx import * 导入
__foo表示类的私用成员(双下划线开头)、
__foo__代表Python里特殊方法的专用标识
__init__()表示该类的构造函数
